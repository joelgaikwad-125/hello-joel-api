# Hello Joel Dynamic API

## Overview
This is a simple Spring Boot REST API.

## Features
- GET → Get current message
- POST → Save new message
- PUT → Update message
- DELETE → Delete message

## API Endpoints

| Method | Endpoint   | Description |
|--------|-----------|------------|
| GET    | /message  | Get message |
| POST   | /message  | Create message |
| PUT    | /message  | Update message |
| DELETE | /message  | Delete message |

## Example

POST Body:
Hello Joel

GET Response:
Hello Joel

## Run
mvn spring-boot:run
